> ## 새로운 데이터베이스 생성하기
> **데이터베이스**(->**스키마**) 파일을 생성한 뒤 작업을 시작한다.
> (+MySQL 같은 프로그램은 SQL문을 작성하지 않아도 체크만 해 주면 SQL문이 자동으로 작성됨)
> ```SQL
> CREATE 데이터베이스명;
> ```
> 1. 테이블 만들고 여러 설정 해주기.
> ```SQL
> CREATE TABLE 테이블명(
>    [칼럼1] [데이터타입] [설정(기본 형식)],
>    [칼럼2] [데이터타입] [설정(기본 형식)],
>    [칼럼3] [데이터타입] [설정(기본 형식)],
>    ...
> )
> ```
